# Ziki AI

Your personal AI assistant — powered by Google Gemini.

## Features
- Chat with Gemini 2.5 Flash
- Streaming responses
- Conversation history (localStorage)
- Dark/light mode
- Mobile responsive

## Setup
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Deploy to Vercel
Add GOOGLE_GENERATIVE_AI_API_KEY in Vercel environment variables.
